# Exemplos de Estruturas de Controle em C#

Este repositório contém exemplos de código em C# para fins didáticos, demonstrando estruturas de controle e lógica de programação fundamentais.

## Conteúdo

O projeto é uma única aplicação de console (`Console App`) onde cada exemplo está comentado para facilitar a execução individual.

1.  **Verificador de Par ou Ímpar**
    *   **Conceito:** Uso da estrutura condicional `if-else` e do operador ternário.
    *   **Técnica:** Demonstra a validação de entrada de usuário com `int.TryParse()` para evitar exceções (`FormatException`).

2.  **Tabuada Simples**
    *   **Conceito:** Geração de uma sequência com o loop `for`.
    *   **Técnica:** Interpolação de strings para formatação da saída.

3.  **Soma e Iteração em Coleções**
    *   **Conceito:** Iteração sobre coleções com o loop `foreach`.
    *   **Técnica:** Demonstra o uso com `Array`, `List` e `Dictionary<TKey, TValue>`.

4.  **Validação de Entrada com Loop**
    *   **Conceito:** Garantir a entrada de dados válidos com um loop `do-while`.
    *   **Técnica:** O loop executa ao menos uma vez, ideal para cenários de "tente novamente".

5.  **Contagem Regressiva**
    *   **Conceito:** Uso do loop `while` para executar um bloco de código enquanto uma condição for verdadeira.
    *   **Técnica:** Controle de fluxo com decremento de uma variável contadora.

6.  **Loops Aninhados**
    *   **Conceito:** Uso de um loop `for` dentro de outro para criar lógicas matriciais.
    *   **Técnica:** Geração das tabuadas de 1 a 5 em uma única execução.

## Como Usar

1.  Clone o repositório.
2.  Abra o projeto em um ambiente de desenvolvimento .NET (Visual Studio, VS Code, etc.).
3.  Descomente o bloco de código que deseja testar.
4.  Execute a aplicação.

# C# Control Structures Examples

This repository contains C# code examples for educational purposes, demonstrating fundamental control structures and programming logic.

## Content

The project is a single Console App where each example is commented out to facilitate individual execution.

1.  **Even or Odd Checker**
    *   **Concept:** Use of the `if-else` conditional structure and the ternary operator.
    *   **Technique:** Demonstrates user input validation with `int.TryParse()` to prevent exceptions (`FormatException`).

2.  **Simple Multiplication Table**
    *   **Concept:** Generating a sequence with a `for` loop.
    *   **Technique:** String interpolation for output formatting.

3.  **Sum and Iteration in Collections**
    *   **Concept:** Iterating over collections with a `foreach` loop.
    *   **Technique:** Demonstrates usage with `Array`, `List`, and `Dictionary<TKey, TValue>`.

4.  **Input Validation with a Loop**
    *   **Concept:** Ensuring valid data entry using a `do-while` loop.
    *   **Technique:** The loop executes at least once, ideal for "try again" scenarios.

5.  **Countdown**
    *   **Concept:** Using a `while` loop to execute a block of code as long as a condition is true.
    *   **Technique:** Flow control by decrementing a counter variable.

6.  **Nested Loops**
    *   **Concept:** Using a `for` loop inside another to create matrix-like logic.
    *   **Technique:** Generates multiplication tables from 1 to 5 in a single run.
